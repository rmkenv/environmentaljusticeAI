## COMMON ISSUES & FIXES

### Issue: "App won't open"
- [ ] Check both files are in same folder
- [ ] Try dragging index.html to browser
- [ ] Try right-click ‚Üí Open With ‚Üí Browser
- [ ] Try Python server method

### Issue: "Blank page loads"
- [ ] Wait 5 seconds for page to load
- [ ] Refresh browser (Ctrl+R or Cmd+R)
- [ ] Check browser console (F12) for errors
- [ ] Try a different browser

### Issue: "API modal doesn't appear"
- [ ] Open browser DevTools (F12)
- [ ] Check console for JavaScript errors
- [ ] Try hard refresh (Ctrl+Shift+R)
- [ ] Clear browser cache

### Issue: "API key not saving"
- [ ] Copy your ENTIRE key (no spaces)
- [ ] Check browser allows localStorage
- [ ] Try a different browser
- [ ] Check for typos in your key

### Issue: "Map not loading"
- [ ] Check internet connection
- [ ] Check browser console for errors
- [ ] Try waiting 5 more seconds
- [ ] Refresh page

### Issue: "AI responses not working"
- [ ] Verify API key is configured (check status badge)
- [ ] Make sure key matches your provider
- [ ] Check you haven't exceeded rate limits
- [ ] Try a different AI provider
