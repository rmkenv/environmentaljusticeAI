## üÜò Troubleshooting

### App won't open
- Try double-clicking `index.html`
- Or use Python: `python -m http.server 8000`
- Make sure both files are in same folder

### API key isn't saving
- Check that you copied the ENTIRE key
- No spaces before/after
- Browser must allow localStorage

### Map not loading
- Check internet connection
- Wait a moment for map to load
- Try a different tab

### Questions not working
- Make sure API key is configured
- Check it's a valid key (not truncated)
- Try a different API provider
